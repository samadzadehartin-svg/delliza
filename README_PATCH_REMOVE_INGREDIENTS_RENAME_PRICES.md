# پچ دلیزا: حذف مواد اولیه، اصلاح اوچکاتمان و قیمت‌ها

این پچ این تغییرات را اعمال می‌کند:

- همه مواد اولیه/ترکیبات محصولات خالی و از نمایش مشتری مخفی شد.
- «کیک اوچکلمان» به «کیک اوچکاتمان» تغییر کرد.
- قیمت کیک براونی شد ۱٬۷۰۰٬۰۰۰ تومان.
- قیمت کیک هویج و گردو شد ۱٬۰۳۸٬۰۰۰ تومان.
- قیمت استرول کیک سیب و دارچین شد ۸۹۰٬۰۰۰ تومان.
- قیمت کیک اوچکاتمان شد ۱٬۴۴۰٬۰۰۰ تومان.
- قیمت کیک موکا شد ۱٬۰۰۰٬۰۰۰ تومان.

اگر Supabase دیتای قبلی را برگرداند، فایل زیر را در SQL Editor اجرا کن:

`supabase/remove-all-ingredients-rename-and-prices.sql`
