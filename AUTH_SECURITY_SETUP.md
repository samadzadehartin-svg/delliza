# راه‌اندازی Supabase Auth و نقش‌های دلیزا

این نسخه ورود پنل مدیریت و پنل فروش را از یوزرنیم/پسورد داخل کد جدا می‌کند و به Supabase Auth وصل می‌کند.

## 1) اجرای SQL نقش‌ها
در Supabase برو به SQL Editor و فایل زیر را اجرا کن:

`supabase/auth-roles-setup.sql`

## 2) ساخت کاربر مدیر
در Supabase برو به Authentication > Users و یک کاربر با ایمیل مدیر بساز.
بعد در همان فایل SQL، بخش آخر را پیدا کن، ایمیل `YOUR_ADMIN_EMAIL@example.com` را با ایمیل مدیر عوض کن و آن insert را اجرا کن.

## 3) ساخت کاربر فروش
برای فروشنده هم در Authentication > Users یک کاربر بساز.
بعد ایمیل `STAFF_EMAIL@example.com` را عوض کن و insert فروشنده را اجرا کن.

## 4) ورود به سایت
از این به بعد:

- `/admin/` با ایمیل و پسورد کاربر دارای نقش `admin` باز می‌شود.
- `/staff/` با ایمیل و پسورد کاربر دارای نقش `staff` یا `admin` باز می‌شود.

نکته: این فایل مرحله اتصال Auth و نقش‌هاست. برای قفل کامل دیتابیس، مرحله بعدی فعال‌سازی policyهای سخت‌گیرانه RLS روی `app_data` است.
