# پچ آپلود عکس حلوای شیر و حلوای خرمایی گردویی

این پچ دو عکس جدید حلوا را به سایت اضافه می‌کند و همچنان اصلاح قیمت دونات/کروسان و حذف B2B/B2C قبلی را حفظ می‌کند.

فایل‌های اصلی اضافه/به‌روزرسانی‌شده:

- assets/images/products/product-33.jpg → حلوای خرمایی گردویی
- assets/images/products/product-34.jpg → حلوای شیر
- assets/js/data.js
- assets/js/products.js
- data/products.json
- supabase/upload-images-and-fix-donut-croissant.sql

اگر سایت بعد از Deploy هنوز عکس قبلی نشان داد، فایل SQL را در Supabase SQL Editor اجرا کنید.
