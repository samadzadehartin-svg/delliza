# پچ عکس محصولات + قیمت دونات و کروسان

این پچ روی نسخه اصلی `delliza-main.zip` ساخته شده است.

تغییرات:
- آپلود عکس‌های جدید برای کروسان، دونات، موچی ساده، موچی اسنیکرز کارامل، موس میوه و چند حلوا سینی.
- حذف نمایش B2B/B2C از سایت و فرم‌ها.
- دونات: ۸۹۰٬۰۰۰ تومان / بسته ۱۰ عددی، حداقل سفارش ۱۰ عدد.
- کروسان: ۹۹۰٬۰۰۰ تومان / بسته ۱۰ عددی، حداقل سفارش ۱۰ عدد.
- محاسبه سبد خرید برای بسته ۱۰ عددی اصلاح شده؛ ۱۰ عدد دونات = ۸۹۰٬۰۰۰ تومان، نه ۸٬۹۰۰٬۰۰۰ تومان.

اگر بعد از Deploy هنوز اطلاعات قبلی آمد، فایل SQL زیر را در Supabase اجرا کن:
`supabase/upload-images-and-fix-donut-croissant.sql`
