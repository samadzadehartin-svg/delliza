# دلیزا - سایت محصولات قنادی

ساختار:
- `/buyer/` پنل مشتری
- `/staff/` پنل فروش
- `/admin/` پنل مدیریت

ورود دمو:
- فروش: `staff123 / staff123`
- مدیریت: `admin123 / admin123`

ویژگی‌ها:
- مدیریت محصول، قیمت، موجودی، دسته‌بندی، عکس، گالری
- فعال/غیرفعال کردن محصول
- فعال/غیرفعال کردن نمایش بخش‌های محصول برای مشتری
- ثبت سفارش مشتری
- مدیریت سفارش در پنل فروش
- تنظیمات برند در مدیریت
- آماده آپلود در GitHub/Vercel
