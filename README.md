# دلیزا - سایت محصولات قنادی

نسخه آماده Vercel برای کاتالوگ و سفارش محصولات دلیزا.

## مسیرها

- `/buyer/` صفحه مشتری و ثبت سفارش
- `/staff/` پنل فروش
- `/admin/` پنل مدیریت

## ورود دمو

- مدیریت: `admin123 / admin123`
- فروش: `staff123 / staff123`

## محصولات

تمام ۵۴ محصول دلیزا داخل `assets/js/data.js` اضافه شده‌اند:

- کیک و چیزکیک
- حلوا سینی
- حلوا تک‌نفره
- کلاسیک
- شیرینی و کوکی
- دسر

## نکته مهم

این نسخه بدون دیتابیس است. تغییرات پنل مدیریت در مرورگر همان دستگاه ذخیره می‌شود. برای ذخیره دائمی آنلاین باید در مرحله بعد دیتابیس یا API اضافه شود.

## Deploy روی Vercel

ریپو را در Vercel Import کن. تنظیم خاصی لازم نیست. فایل `vercel.json` صفحه اصلی را به `/buyer/` هدایت می‌کند.
